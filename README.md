# Portafolio — Ronald Trejo

Portafolio personal desarrollado con React, TypeScript y Tailwind CSS. Presenta mis proyectos, stack tecnológico, trayectoria y datos de contacto.

## Tecnologías

- **React 19** + **TypeScript**
- **Tailwind CSS v4**
- **Framer Motion** — animaciones y transiciones
- **Vite** — bundler y servidor de desarrollo

## Características

- Modo oscuro / claro con persistencia
- Animaciones de scroll con reveal progresivo
- Cursor personalizado
- Diseño completamente responsivo
- Secciones: Hero, Proyectos, Stack, Sobre mí, Contacto

## Instalación

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

El output queda en `dist/` listo para desplegarse en GitHub Pages o cualquier hosting estático.

## Despliegue

El sitio se despliega automáticamente en GitHub Pages en cada push a `main` mediante GitHub Actions.
